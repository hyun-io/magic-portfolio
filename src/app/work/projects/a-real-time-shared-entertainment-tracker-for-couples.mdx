## Overview

A full-stack entertainment tracker built for long-distance couples to share movies, TV shows, and games, featuring real-time updates, chat, and collaborative commenting. Designed with a modern dark theme and optimised for both desktop and mobile.

## Key Features

- **Real-time Synchronisation**: Instant updates across both users using Supabase real-time subscriptions, ensuring that both partners see the same content and interactions without needing to refresh.
- **Interactive Dashboard**: Tab-based navigation for Movies, TV Shows, and Games. Each tab displays a list of shared entertainment items with options to add, edit, and comment.
- **Dual View Modes**: Collaborated closely with designers by setting up a shared design library in Figma. This library was synchronized with the codebase, ensuring design handoffs were seamless and that design tokens remained consistent across both platforms.
- **Smart Episode Tracking**: Track TV show progress with S1E2 format and season/episode management, allowing couples to easily keep track of their shared entertainment experiences.
- **Collaborative Comments**: Full CRUD comment system with star ratings and real-time updates, enabling couples to share their thoughts and feedback on their entertainment choices.
- **Integrated Chat**: Persistent chat panel with message history, deletion, and timestamps.
- **Online Status Indicators**:  Real-time user presence with visual status badges, enhancing the sense of connection between users.
- **Notification System**: Custom toast notifications for all user actions, providing immediate feedback and enhancing the user experience.
- **Dark/Light Mode**: Theme persistence with smooth transitions, allowing users to choose their preferred visual style.
- **Responsive Design**: Mobile-first approach with consistent experience across devices.

### Key Contributions
- Architected a complete real-time system using Supabase subscriptions for instant cross-user updates
- Built an intuitive episode tracking interface for TV shows with manual progress adjustment
